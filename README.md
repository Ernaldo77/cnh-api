# CNH-API

Uma API RESTful desenvolvida em Flask voltada para CNHs.\
Nela é possível fazer todo o CRUD, permitindo criar, ver, atualizar e deletar carteiras de motorias.

## Preparação

Após baixar o projeto, é necessario instalar o Flask para que a aplicação funcione.\
Para isso, siga esses passos:

### Crie um Ambiente Virtual.
```bash
python -m venv venv
```

### Entre no Ambiente.
Windows:
```bash
venv/Scripts/activate
```
Linux:
```bash
source venv/bin/activate
```

### Instale o Flask:
```bash
pip install flask
```

### Rode o Projeto:
```bash
python app.py
```
